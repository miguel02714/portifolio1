Neurabot - IA Inteligente

Este é o Neurabot, um projeto de inteligência artificial criado por mim. Desenvolvi toda a lógica, backend, e integração usando Python. A estrutura visual foi feita com HTML e CSS, e utilizei JavaScript para deixar a interface mais interativa.

A IA possui integração com APIs externas, login com Google, sistema de autenticação com MySQL e uma arquitetura própria para recuperação de dados. É um sistema RAG completo.

Criado por um desenvolvedor de 14 anos com paixão por IA e backend.
